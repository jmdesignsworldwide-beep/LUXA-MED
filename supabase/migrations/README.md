# Migraciones de Supabase

Cada migración es un archivo `.sql` con prefijo de timestamp, p. ej.
`20260622_init.sql`. Se versionan aquí para tener historial; se aplican vía la
Supabase Management API usando un Personal Access Token temporal (ver
[`../../docs/SETUP.md`](../../docs/SETUP.md), sección 3).
